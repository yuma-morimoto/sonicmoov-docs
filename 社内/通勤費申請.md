
通勤費は月末日付で1件にまとめて申請する。

## 入力内容

| フィールド | 値 |
|---|---|
| `date_hassei` | 申請対象月の末日（例: 2026-04-30） |
| `id_shubetsu` | 4（販管） |
| `id_himoku` | 52（通勤費／販管） |
| `memo1` | ー |
| `memo2` | 通勤費（京王堀之内〜神谷町） |
| `memo3` | 出社X日（X = 出社日数。片道のみの日は0.5日換算） |
| `zeikomi_kingaku` | 往復1,194円 × 出社日数（片道597円 × 2）。片道のみの日は597円で換算 |
| `is_kazei` | 1（税込） |
| `id_master_shouhizei` | 5（10%） |
| `id_type_seikyusho` | 1（適格請求書（領収書）） |
| `id_expense_request_type` | 100（立替） |

## 金額の計算例

- 出社3日（すべて往復）: 1,194円 × 3 = 3,582円
- 出社2日（往復）+ 片道1回: 1,194円 × 2 + 597円 = 2,985円
  - この場合 `memo3` は「出社2.5日」と記載
