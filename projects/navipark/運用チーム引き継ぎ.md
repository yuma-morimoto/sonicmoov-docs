
## プロジェクト概要

- ナビパーク駐車場の時間貸決済・月極契約をアプリ上から行えるサービス
- 時間貸決済：駐車場の決済をアプリ上からできる機能（クレジットカードや PayPay など現金以外の決済にも対応）
- 月極契約：駐車場の月極契約をアプリ上から申請できる機能
- フロントが担当する部分：
	- **アプリ**：Nuxt 3 + Vite + TypeScript の SPA。Flutter アプリの WebView 上で表示される
	- **管理画面**：Laravel の MVC 上で Nuxt が SPA として表示される。Blade ファイルのみフロント側で管理が必要

## テスト自動化の現状

### 単体テストについて
- composables 配下にvitestのテストコードがあります。
- vitest browser modeでコンポーネントテストも行なっています。
### E2Eテストについて
- Playwright と Page Object Model で設計しています。
- まだテスト量が不十分なのと動作が不安定になる場合があるので注意してください。
