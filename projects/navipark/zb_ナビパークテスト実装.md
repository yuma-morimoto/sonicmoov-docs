## 環境構築

- Playwrightに使用するソフトウェアをインストールする
```bash

npx playwright install

```

## ディレクトリ構成

```
e2e/
- pages/ - ページオブジェクトを格納します
- base/ - 基底クラスや共通ユーティリティを配置します
- selectors/ - セレクタの定義ファイルを管理します
- components/ - 再利用可能なUIコンポーネントを配置します
- specs/ - テストケースファイルを配置します
```

## テスト項目

### 貸出管理
- 仮申込（個人/法人）
- 本申込（個人/法人）
- 駐車場詳細
- 貸出契約一覧
- 貸出決済
- 決済履歴

### それ以外
- 駐車場詳細（時間貸し）
- 駐車場検索
- マップ表示

## 駐車場詳細のテスト項目

### 1. 基本的な情報確認ジャーニー

1. ユーザーが駐車場詳細ページにアクセス
2. 駐車場情報（名前、住所、料金等）を確認
3. 時間貸駐車場と月極駐車場のタブを切り替えて情報を比較
4. 必要な情報を得て他のページに移動

### 2. 駐車場決済ジャーニー（時間貸し）

1. ユーザーが駐車場詳細ページにアクセス
2. 利用可能な車庫を確認
3. 車庫選択ボタンをクリック
4. 車庫番号選択ダイアログで希望の車庫を選択
5. 決済方法選択画面で支払い方法を選択
6. 決済確認画面で内容を確認
7. 決済を実行
8. 決済完了画面で結果を確認

### 3. 定期利用券購入ジャーニー

1. ログインユーザーが駐車場詳細ページにアクセス

2. 定期利用券の情報を確認

3. 購入可能な定期利用券を選択

4. 定期利用券購入確認画面で内容を確認

5. 購入を実行

6. 購入完了画面で結果を確認

  

### 4. エラー対応ジャーニー

  

1. ユーザーが削除済み駐車場にアクセス

2. 削除済み駐車場ダイアログが表示される

3. ダイアログを閉じてホームページにリダイレクト
